À propos

Escape game virtuel développé pour la plateforme Discord (outil de discussion instantanée). Jeu pour 4 à 24 joueurs simultanés. Chaque partie peut être adaptée en fonction de l'âge, de l'expérience et du nombre de joueurs.

Matériel nécessaire pour jouer

Pour participer, une connexion internet suffisante est nécessaire. Un casque avec micro est recommandé pour une communication vocale de qualité. Un ordinateur est préférable pour une expérience de jeu optimale, mais un terminal mobile est suffisant. Dans ce dernier cas, il faudra installer l'application mobile Discord.
